## üöÄ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- OpenAI API key (optional, for AI-powered generation)

### Installation
1. Clone or download this repository
2. Open `index.html` in your web browser
3. (Optional) Configure your OpenAI API key for AI-powered generation
4. Start generating brands!

### OpenAI API Setup
1. **Get an API Key**: Visit [OpenAI Platform](https://platform.openai.com/api-keys) and create an API key
2. **Configure in App**: Click the settings gear icon in the header
3. **Enter API Key**: Paste your API key (starts with `sk-`)
4. **Test Connection**: Use the "Test Connection" button to verify
5. **Save & Enable**: Click "Save & Enable AI" to activate AI generation
